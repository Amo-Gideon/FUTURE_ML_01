## Project Overview
This project builds a sales forecasting system using the Superstore dataset.
Goal: Predict future sales to help businesses plan inventory, staffing, and cash flow.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Steps
1. Load and clean data
2. Analyze monthly sales trends
3. Create time-based features
4. Build forecasting model
5. Predict 6 months of future sales
6. Visualize results for business users

## File Structure
- data/: Dataset (Sample - Superstore.csv)
- notebooks/: Jupyter code (main.ipynb)
- output/: All charts and predictions
- README.md: Project explanation
- requirements.txt: Libraries needed

## How to Run
1. Install libraries:
pip install -r requirements.txt
2. Open notebooks/main.ipynb
3. Run all cells
